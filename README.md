# 校园论坛系统服务端

**环境：Node.jsV16.0**

**数据库：MySQL 8.0**

**说明：**

**项目启动之前需要按照以下步骤进行配置：**

> 1. 在本地mysql中，创建一个名为bbs的数据库，随后导入sql文件，创建完成后在admin表中添加一个管理员账号，方便登录；
> 2. 修改app/config/db.js  文件中的数据库配置信息；
> 3. 修改app/routes/app/user.js 中的邮件配置，修改为自己的邮箱；

配置完成后执行以下命令

1. 安装依赖

   `npm i`

2. 启动项目

   `npm start`
